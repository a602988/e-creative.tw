{[(name)]}最早起萌於 2008 年的盛夏，我們是一家充滿新鮮創意及設計熱情的網路公司。 ＂臻至完美、近乎苛求＂恰如其分的為{[(name)]}寫實了經營理念，多年以來，我們培育與精鍊出了一群優秀聲譽、 頂尖質量的好手，並且橫跨多方面領域的設計團隊，我們擅於打造以創新為本的設計作品、 專業於多種領域的軟體工程技術及全方位的服務導向整合，{[(name)]}總是企圖將創意發揮到淋漓盡現， 並且努力使命必達，我們深信知識與經驗的累積才是打動人心的服務。

然而在{[(name)]}，我們對於這裡的所有成員，無論他們所負責的部門職責為何， 都期待大家具備創意思維和解決方案。創新的形式有許多種， 但我們的開發人員總是能在不斷的腦力激盪下創造出無限可期的精彩新意。 對於設計美學的堅持，以及在這裡所面對的每一個專案執行，我們希望將品牌設計理念傳達給消費者， 設計是無形的品味，更是具體的視覺表現，如何賦予創意無限的可能性， 如何從『有形的生產消費』轉變成『無形的生活價值』，才是我們追崇的夢想種子。

用心傾聽客戶的需求，以獨一無二的風格，全然不凡的質感，為客戶開創嶄新的品牌價值與尊嚴。 更以不斷突破的網路技術，令人驚艷的領航設計，來為我們的客戶服務。在這個過程中， 我們堅持只保留最完美的部份給我們的客戶，而且我們總是努力不懈的設法超越客戶的期待， 因為我們知道你需要什麼！

團隊介紹

創意總監 Creative Director Eric Chang

創意，源自於瞬間的感動，真心的投入就會非常有感覺，用心聆聽客戶的需求，從中捕捉最真實的”感動”。 善於以專業精闢的規劃能力與豐富的網路經驗，打造出企業與品牌嶄新的形象與創意的無限魅力。

設計總監 Design Director Jean Lin

人類漫長的歷史中，所謂設計美學時刻在經歷著各種衍變和流派的洗禮，我們也很難去評論哪種風格是最優秀的， 事實上這些風格和流派總是因人而異，或許，正因為如此，設計只有經典，而卻無法成為永恒。

技術總監 Technical Director Joe Lin

面向技術，往往不是只有深度，還需要有足夠的廣度，當面對看似艱澀難懂的複雜問題時，或許只要轉個彎， 就會豁然開朗得找到出口。其實越是簡單越有力量！如何化繁為簡、駕簡馭繁才是智慧的核心價值。

專案經理 Project Manager Ava Xiao

憑藉著女性細膩獨到的眼光，專注於探究客戶的產業與市場需求，擬定完整網路商業策略及專案規劃, 曾經協助許多中小企業建立獨樹一格的品牌形象與嶄新風格, 我們始終相信客戶的成功才是我們最大的成功。

總設計師 Chief Designer Mia Liu

善於以豐富的色彩美學與富有設計感的藝術天份，打破傳統的框架思維，憑著優異的美學見解， 提供給使用者愉悅舒適的瀏覽體驗，擅以將繁複的架構整理出有跡可循的脈絡，以打造出令人愛不釋手的作品。

資深設計師 Senior Designer Nomi Zhuang

設計提供的除了是美學之外還應該提供人性化功能，只要能夠多投入一點心靈上的巧思，設計便得以更深入人心， 才更能值得稱為佳作，喜歡參與團隊間的腦力激盪，在理性的探討之下，讓設計作品琢至完美境界。

網頁設計師 Web Designer Tina Liou

喜歡運用多元符號，來塑造設計的獨特性，經常涉獵各類型的文化題材與元素，並加以重新詮釋， 來突破單一手法的市場慣性設計。以圖層與透明度營造縱深，自然帶出層次分明的空間感，也讓一切更引人入勝。

網頁設計師 Web Designer Mandy Chen

源至於對設計與藝術的熱愛，憑藉熱忱與務實的學習態度，逐步累積實力並摸索出獨創之風格。 善於運用明快清晰的現代設計技巧鋪排，以及充滿令人驚喜的創意設計，在美感與商業需求之間的取得完美平衡。

總工程師 Chief Engineer Daniel Chen

有時候我們會因為害怕而不敢突破，希望能藉著正面的力量，讓我們的團隊能擁有屬於自己表現的空間。 當看見大家的努力與突破，當每個人願意給自己機會去嘗試的時候，那份成長才是我們能真實地感受到的！

網頁工程師 Web Engineer Plane Zhao

如何打造出高品質使用者體驗的產品一直以來是團隊共同的目標，身為團隊中的一員除了感到興奮之外， 還讓我學習到協力開發的重要性，因為除了享受設計細節的工作之外，我們還更在意使用者體驗的感覺。
